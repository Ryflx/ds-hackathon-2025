Demo Sidekick Test Page

Use this page to test the Demo Sidekick Chrome extension banner injection.

Testing Instructions

  1. Load the Demo Sidekick extension in Chrome
  2. Click on the extension icon in the toolbar
  3. Configure a demo with:
    • Demo Type: IAM or CLM
    • Persona Name: Any name you like
    • Demo Flow: Select one of the available flows
  4. Click "Configure Demo" to inject the banner
  5. You can also use the "Test Banner" button for quick testing

Expected Behavior

When the demo is configured, you should see:

  • A banner appearing at the bottom of this page
  • The banner should show persona information, demo tagline, and flow details
  • A close button (Ã—) to end the demo
  • Console logs showing the injection process

Debugging

If the banner doesn't appear:

  • Open Developer Tools (F12)
  • Check the Console tab for "Demo Sidekick" messages
  • Refresh this page and try again
  • Make sure the extension has proper permissions
